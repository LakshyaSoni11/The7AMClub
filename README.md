# THE 7AM CLUB | Community & Fitness Platform

A high-performance, premium web platform built for **The 7AM Club**, Raipur's most inclusive fitness and hustle community. This project features a state-of-the-art interactive design, glassmorphism aesthetics, and a fully responsive experience across all devices.

![Project Preview](https://images.unsplash.com/photo-1552674605-db6ffd4facb5?q=80&w=2070&auto=format&fit=crop)

## 🚀 Key Features

- **Interactive Particle System**: A high-performance background text effect that reacts to user movement and mouse interaction. Optimized for both desktop and mobile.
- **Responsive Navigation**: A modern, fixed-position navbar that transitions into a full-screen mobile menu for seamless navigation.
- **Stacked Gallery (Mobile First)**: A custom interactive "Card Stack" gallery designed specifically for mobile users, allowing them to swipe through community memories.
- **3D Mindset Library**: An interactive 3D book component built to showcase community-recommended literature (e.g., *Atomic Habits*).
- **Glassmorphism UI**: A consistent design system using blurred backdrops, neon accents, and vibrant gradients for a premium feel.
- **Optimized Performance**: Smooth 60FPS animations achieved through custom canvas optimizations and efficient CSS blurs.

## 🛠️ Technology Stack

- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS 4 (CSS-based configuration)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Design System**: Shadcn/UI (Customized)
- **Font**: Syne, Inter, Outfit, and Geist Variable

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/LakshyaSoni11/The7AMClub.git
   cd The7AMClub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📱 Mobile Optimization

This platform has been rigorously refined for mobile devices:
- **Centered Content**: All sections are perfectly balanced for vertical mobile viewing.
- **Touch Targets**: Increased button sizes and interactive areas for better UX.
- **Performance Tiers**: Automatically detects device type to adjust animation complexity, ensuring no lag on older smartphones.
